REFERENTIECommandoreferentie

Alle slash commands in Claude Code, gegroepeerd op functie. De essentiële commando's zijn vetgedrukt.

Sessie & Context

CommandoWat het doet
/clearWis context, behoud CLAUDE.md (aliases: /reset, /new)
/compact [focus]Comprimeer geschiedenis met optionele focus-instructie
/contextVisuele grid van huidig tokengebruik
/fork [naam]Fork het gesprek op dit punt
/resume [sessie]Hervat een eerder gesprek (alias: /continue)
/rename [naam]Hernoem de sessie, of laat automatisch naamgeven
/export [bestand]Exporteer gesprek als plain text
/exitSluit af (alias: /quit)

Model & Modus

CommandoWat het doet
/model [model]Wissel model (sonnet, opus, haiku) met effort level
/planActiveer Plan Mode -- analyseren zonder wijzigen
/fast [on|off]Toggle snellere output (zelfde model)

Code & Git

CommandoWat het doet
/initGenereer starter CLAUDE.md op basis van je project
/diffInteractieve diff viewer (uncommitted + per-turn changes)
/rewindSpoel terug naar eerder punt in gesprek en code (alias: /checkpoint)
/pr-comments [PR]Haal GitHub PR comments op (auto-detecteert of accepteert URL/nummer)
/install-github-appZet Claude GitHub Actions app op voor een repository
/security-reviewAnalyseer pending changes op security kwetsbaarheden
