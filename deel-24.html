DEEL 24MCP-servers -- Claude verbinden met de buitenwereld

Het Model Context Protocol (MCP) is een open standaard waarmee Claude Code verbinding maakt met externe tools en diensten. Waar plugins alles bundelen (skills, hooks, agents), zijn MCP-servers gerichte koppelingen: een browsertool, een database-client, een API-connector.

Wat is een MCP-server?

Een MCP-server is een klein programma dat draait op je machine en Claude extra tools geeft. Denk aan: een browser besturen, bestanden doorzoeken, API's aanroepen, of databases bevragen. Claude "ziet" deze tools automatisch en kan ze gebruiken in je gesprek.

PLUGINS

Bundels

Combineren skills, hooks, agents en MCP-servers in een pakket. Installeer via /plugin.

MCP-SERVERS

Gerichte tools

Enkele koppeling met een externe dienst. Configureer via claude mcp add.

MCP-server toevoegen

# Voeg een MCP-server toe (standaard: user scope)
claude mcp add server-naam -- npx -y @pakket/mcp-server

# Voorbeeld: Playwright (browser besturen)
claude mcp add playwright -- npx -y @anthropic-ai/mcp-server-playwright

# Voorbeeld: Filesystem (bestanden buiten je project)
claude mcp add filesystem -- npx -y @anthropic-ai/mcp-server-filesystem /pad/naar/map

# Bekijk welke servers actief zijn
claude mcp list

# Verwijder een server
claude mcp remove server-naam

Scopes

MCP-servers kunnen op verschillende niveaus worden geconfigureerd:

ScopeWaarWanneer
user~/.claude/settings.jsonAltijd beschikbaar, in elk project
project.claude/settings.jsonAlleen in dit project, gedeeld met team
local.claude/settings.local.jsonAlleen voor jou, niet in Git
 
# Stel de scope in bij het toevoegen
claude mcp add --scope user playwright -- npx -y @anthropic-ai/mcp-server-playwright
claude mcp add --scope project database -- npx -y @mcp/postgres-server

Voorbeeld: Playwright in actie

Met de Playwright MCP-server kan Claude je browser besturen. Handig voor frontend-debugging, visual testing, of data ophalen van websites.

# Na installatie kun je in je gesprek zeggen:
"Open de homepage en maak een screenshot"
"Vul het contactformulier in met testdata"
"Check of de navigatie werkt op mobiel formaat"
"Vergelijk de live site met mijn lokale versie"

Voorbeeld: claude.ai MCP-koppelingen

In claude.ai (de webversie) kun je ook MCP-koppelingen activeren, zoals Gmail en Google Calendar. Deze werken via OAuth en geven Claude toegang tot je mail en agenda -- handig voor planning en communicatie.

Eigen MCP-server bouwen

Een MCP-server is een programma dat het MCP-protocol implementeert. Je kunt er zelf een bouwen in Python, TypeScript of elke andere taal:

# Minimaal voorbeeld (TypeScript)
import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio.js";

const server = new McpServer({ name: "mijn-tool", version: "1.0.0" });

server.tool("zoek_klant", { naam: z.string() }, async ({ naam }) => {
  const klant = await database.zoek(naam);
  return { content: [{ type: "text", text: JSON.stringify(klant) }] };
});

const transport = new StdioServerTransport();
await server.connect(transport);
# Registreer je eigen server
claude mcp add mijn-tool -- node /pad/naar/mijn-server.js
Wanneer MCP vs Plugin?

MCP-server: Je wilt een specifieke externe tool koppelen (database, browser, API).
Plugin: Je wilt een complete workflow met skills, hooks en agents bundelen.
Veel plugins gebruiken MCP-servers onder de motorkap -- de plugin regelt dan de configuratie voor je.

Veiligheid

MCP-servers draaien lokaal met jouw rechten. Een database-server heeft toegang tot je database, een filesystem-server tot je bestanden. Geef alleen toegang tot wat nodig is en review welke tools een server aanbiedt (claude mcp list).
