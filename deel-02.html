DEEL 2De Gereedschapskist

Claude Code — Terminal-tool van Anthropic. Leest bestanden, voert commando's uit, schrijft code, doet git. $20/mnd (Pro) of $100/mnd (Max). Dit is waar het hele systeem uit deze gids op draait.

Alle surfaces

Claude Code draait niet alleen in je terminal. Dezelfde engine, dezelfde CLAUDE.md, dezelfde settings -- overal.

Terminal CLIVolledige feature set, piping, scriptingGeen visuele diffs
VS Code / CursorInline diffs, @-mentions, plan reviewExtensie vereist
JetBrainsIntelliJ, PyCharm, WebStorm pluginBeta
Desktop appVisuele diff review, meerdere sessies naast elkaarmacOS / Windows
Web (claude.ai/code)Geen lokale setup, lange taken, parallel werkBrowser vereist
Chrome extensieLive web apps debuggen vanuit browserChrome only
Slack (@Claude)Bug report in teamchat → PR (Pull Request) terugSlack app setup

Sessies verplaatsen

Sessies zijn niet gebonden aan een surface. Zet terminal-sessie voort op telefoon met /remote-control. Open in Desktop app met /desktop.

Cursor — AI-powered code-editor (VS Code fork). Agent mode + rules systeem. Windsurf — Vergelijkbaar, met "Cascade" agent.

Aanbeveling

Begin met Claude Code in terminal — alle concepten (CLAUDE.md, skills, hooks, agents) hebben daar native ondersteuning. VS Code extensie of Desktop app ernaast voor visuele diffs.

DEEL 2BEssentiële Commando's

Je hoeft niet elk commando te kennen. Dit zijn de commando's met de meeste impact.

/init

Genereer starter CLAUDE.md op basis van je project

/plan of Shift+Tab ×2

Plan Mode — Claude plant maar codeert niet

/clear

Reset context. Gebruik tussen ongerelateerde taken

/compact [focus]

Comprimeer geschiedenis. Optioneel: focus-instructie meegeven

/model sonnet of opus

Wissel model zonder herstart

/fast

Toggle snellere output (zelfde model, hogere snelheid)

/context

Bekijk huidig tokengebruik als visuele grid

/diff

Interactieve diff viewer met uncommitted en per-turn changes

/cost en /usage

Token-statistieken en plan/rate limit status

/rewind

Spoel terug naar een eerder punt in gesprek en code

/export

Exporteer het gesprek als plain text (voor handoff)

/hooks

Configureer hooks interactief

/permissions

Beheer welke tools Claude mag gebruiken

/doctor

Diagnose installatie en configuratie

/config

Instellingen: thinking mode, output style

Esc Esc

Undo -- rewind de laatste actie van Claude

/rename

Hernoem sessie om later terug te vinden

! <command>

Voer bash-commando direct uit zonder Claude-interpretatie

Tip

Zet in /config thinking mode op true en output style op Explanatory. Dan zie je hoe Claude redeneert. Alle commando's: zie Commandoreferentie.

DEEL 2CCLI als Bouwsteen

Claude Code volgt de Unix-filosofie: je kunt het pipen, scripten en combineren met andere tools. Met de -p (print) flag draait Claude non-interactief -- ideaal voor automatisering.

Piping

# Monitor logs en krijg alerts
tail -f app.log | claude -p "Slack me if you see anomalies"

# Review alleen gewijzigde bestanden
git diff main --name-only | claude -p "review these for security issues"

# Bulk vertaling in CI
claude -p "translate new strings into French and raise a PR"

Scripting & automatisering

# Gestructureerde output met JSON schema
claude -p "list all TODOs" --output-format json --json-schema '{...}'

# Budget limiet voor CI runs
claude -p "fix lint errors" --max-budget-usd 5

# Read-only analyse (geen writes)
claude --allowedTools "Read,Glob,Grep" -p "explain the auth flow"

# Fallback model bij overbelasting
claude -p "write tests" --fallback-model haiku
Krachtig patroon

Combineer -p met --output-format json voor machine-leesbare output. Ideaal voor integratie in CI/CD pipelines, cron jobs, of eigen tooling.
