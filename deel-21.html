DEEL 21Jouw Startplan

WEEK 1–2 · FOUNDATION

Installeer Claude Code + eerste CLAUDE.md

Draai /init. Verfijn: overview, stack, 5-10 regels. Maak ~/.claude/CLAUDE.md. Oefen met /plan, /clear, /compact.

WEEK 3–4 · LITTEKENS

Verzamel je eerste regels

Elke fout → regel. Maak docs/refs/. Commit na elke wijziging. Schrijf handoff files.

WEEK 5–6 · HOOKS

Eerste harde garanties

Pre-commit hook (lint/types). Post-write hook (tests). Configureer via /hooks.

WEEK 7–8 · SKILLS

Eerste herbruikbare workflow

Taak die je 3x handmatig deed → skill in .claude/skills/. Test op consistentie.

WEEK 9–10 · AGENTS

Eerste gespecialiseerde agent

Audit-agent voor één aspect: links, SEO, content freshness.

DOORLOPEND

Feedback loop

Fout → regel. Herhaalde taak → skill. Terugkerende controle → agent. Snoei CLAUDE.md. Experimenteer met modellen en parallel werk.
