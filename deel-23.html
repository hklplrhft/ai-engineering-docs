DEEL 23Plugins, Tools & Community Resources

Claude Code heeft een plugin-systeem waarmee je kant-en-klare extensies kunt installeren. Plugins bundelen skills, hooks, agents en MCP servers.

Plugins installeren

# Open de plugin manager
/plugin

# Installeer een plugin uit de officiële marketplace
/plugin install plugin-naam@claude-plugins-official

# Voeg een community marketplace toe
/plugin marketplace add anthropics/claude-code

# Herlaad na installatie
/reload-plugins

Veiligheid

Plugins draaien code op je machine met je rechten. Installeer alleen plugins van bronnen die je vertrouwt. Check de homepage en code van een plugin voor je installeert.

Zo begin je

Stap 1: Installeer de code intelligence plugin voor je taal (/plugin install typescript-lsp@claude-plugins-official).
Stap 2: Installeer github plugin als je met GitHub werkt.
Stap 3: Bekijk Superpowers of Compound Engineering voor workflow-skills.
Stap 4: Browse de awesome-claude-code lijst voor meer.
