Keybindings aanpassen

Wil je sneltoetsen anders? Bewerk ~/.claude/keybindings.json of gebruik /keybindings:

# Open de keybindings editor
/keybindings

# Of bewerk handmatig
# ~/.claude/keybindings.json
{
  "bindings": [{
    "context": "Chat",
    "bindings": {
      "ctrl+enter": "chat:submit",     // Ctrl+Enter om te versturen
      "enter": "chat:newline"           // Enter voor nieuwe regel
    }
  }]
}
